# Abc-map.fr

The new version of Abc-map's website. Work in progress.
Based on Spring Boot.

## Launch a demo

	$ git clone https://github.com/remipassmoilesel/abcmap-fr
	$ cd abcmap-fr
	$ ./mvnw spring-boot:run

After visit http://localhost:8085 .

## Sass processing

Need Node JS.

	$ npm install
	$ npm install -g gulp
	$ gulp
